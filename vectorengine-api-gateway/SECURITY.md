# Security policy

## Supported versions

This repository currently holds **documentation and metadata** for the Vector Engine API gateway. For security issues affecting the **live API or console**, report through the channels below regardless of this repo’s tag.

## Reporting a vulnerability

Please **do not** open a public GitHub issue for undisclosed security problems.

- **Preferred:** Telegram [**@ZXCZCI**](https://t.me/ZXCZCI) — describe the issue without sharing live API keys.  
- **Backup:** email [**yeallen441@gmail.com**](mailto:yeallen441@gmail.com?subject=%5BVector%20Engine%20security%5D%20disclosure) — subject should start with **`[Vector Engine security]`** so we can triage. Do not attach keys or secrets.

Include:

- Affected host (`api.vectorengine.ai`, console, etc.)  
- Steps to reproduce (redacted)  
- Impact assessment if known  

We aim to acknowledge valid reports within a reasonable window; critical fixes depend on upstream and platform rollout.

## Safe use

- Rotate API tokens if exposed.  
- Never paste `sk-` keys into issues, chats, or screenshots.
